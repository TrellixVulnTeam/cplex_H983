Benders Algorithm and Decompositions in the CPLEX Callable Library (C API)

These routines are available in the CPLEX Callable Library (C API) to support Benders algorithm and decompositions in your applications.

For corresponding routines for use with a remote object, see also Remote Object in the CPLEX Callable Library (C API) Reference Manual

  • CPXXbendersopt and CPXbendersopt
    At any time after a mixed integer program has been created by a call to CPXXcreateprob/CPXcreateprob, the routine CPXXbendersopt/CPXbendersopt may be used to find a solution to that problem using a Benders decomposition algorithm.
