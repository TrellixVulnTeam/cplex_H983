CPXXgetbranchnosolncallbackfunc and CPXgetbranchnosolncallbackfunc

This routine accesses the user-written callback to be called during MIP optimization when branching when no solution is available.

int CPXXgetbranchnosolncallbackfunc( CPXCENVptr env, int(CPXPUBLIC **branchnosolncallback_p)(CPXX_CALLBACK_BRANCH_ARGS), void ** cbhandle_p )

int CPXgetbranchnosolncallbackfunc( CPXCENVptr env, int(CPXPUBLIC **branchnosolncallback_p)(CALLBACK_BRANCH_ARGS), void ** cbhandle_p )

Description

This routine accesses the user-written callback to be called during MIP optimization when instead of the branch callback when no solution is available. See the documentation of CPXXsetbranchnosolncallbackfunc and CPXsetbranchnosolncallbackfunc for a list of conditions in which this user-written callback will be called and for documentation of the user-written callback in Callback Description.

Arguments

Name Description
env

A pointer to the CPLEX environment, as returned by CPXXopenCPLEX/CPXopenCPLEX.

branchnosolncallback_p

A pointer to a user-written callback. If the callback is set to NULL, no callback can be called during optimization.

cbhandle_p

A pointer to user private data. This pointer is passed to the callback.

Return

The routine returns 0 (zero) if successful or an error code otherwise.
