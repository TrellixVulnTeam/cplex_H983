Removed features CPLEX V12.8.0

These release notes announce features that were deprecated in a previous release and have been removed from IBM CPLEX V12.8.0.

Utility to convert file formats removed

Previous versions of CPLEX distributed convert, a utility to convert fixed-format MPS, BAS, or ORD files into a format that CPLEX supports even when the file lacked names of bounds or righthand sides, or when the file contained blank space within a name, blank lines, or uncommented characters. The convert utility was deprecated in CPLEX V12.6.3. It has now been removed.

Python API

The method cplex.terminate has been removed. That method was deprecated in CPLEX V12.7.1 in favor of the Aborter class. For an example that uses an instance of the Aborter class instead of the method cplex.terminate, see mipex4.py, distributed with the product.

The class PresolveCallback was deprecated in a previous release (CPLEX V12.7.1), and it has been removed from CPLEX V12.8.

The following methods were deprecated in a previous release (CPLEX V12.7.1), and they have been removed from CPLEX V12.8:
  • Cplex.solution.basis.get_col_basis
  • Cplex.solution.basis.get_row_basis

  • Cplex.linear_constraints.advanced.add_lazy_cuts
  • Cplex.linear_constraints.advanced.free_lazy_cuts
  • SolveCallback.set_vectors
