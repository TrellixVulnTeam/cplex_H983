Remote Object (annotations) in the CPLEX Callable Library (C API)

These routines are available in the CPLEX Callable Library (C API) for you to annotate your model in an application of the remote object. An annotation associates an ordered pair of key and value with an object in your model. You can use these annotations, for example, to specify blocks in a Benders decomposition of your model.

  • CPXXdeldblannotation_multicast and CPXdeldblannotation_multicast
    The routine CPXXdeldblannotation_multicast/CPXdeldblannotation_multicast deletes a double valued annotation.
  • CPXXdeldblannotations_multicast and CPXdeldblannotations_multicast
    Multicast version of CPXXdeldblannotations_multicast/CPXdeldblannotations_multicast.
  • CPXXdellongannotation_multicast and CPXdellongannotation_multicast
    The routine CPXXdellongannotation_multicast/CPXdellongannotation_multicast deletes a long valued annotation.
  • CPXXdellongannotations_multicast and CPXdellongannotations_multicast
    Multicast version of CPXXdellongannotations_multicast/CPXdellongannotations_multicast.
  • CPXXreadcopyannotations_multicast and CPXreadcopyannotations_multicast
    Multicast version of CPXXreadcopyannotations/CPXreadcopyannotations.
  • CPXXsetdblannotations_multicast and CPXsetdblannotations_multicast
    The routine CPXXsetdblannotations_multicast/CPXsetdblannotations_multicast sets the annotation values for a set of modeling objects in the CPLEX problem object.
  • CPXXsetlongannotations_multicast and CPXsetlongannotations_multicast
    The routine CPXXsetlongannotations_multicast/CPXsetlongannotations_multicast sets the annotation values for a set of modeling objects in the CPLEX problem object.
