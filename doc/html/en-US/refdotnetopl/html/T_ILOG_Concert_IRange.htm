Remarks

lb and ub are double values, referred to as the lower bound and upper bound of the ranged constraint, and expr is an expression. Values +- infinity can be used as bounds. This allows you to use IRange objects to represent more commonly used constraints:

  • for expr == rhs, set lb = ub = rhs
  • for expr <= rhs, set lb = -infinity and ub = rhs
  • for expr >= rhs, set lb = rhs and ub = infinity

Through the selection of lb and ub, constraints usually written as expr ~ rhs with ~ in <=, >=, or == can also be expressed. IRange objects can be constructed using a variety of methods defined for IModeler: AddRange(), AddEq(), AddLe(), AddGe(), Range(), Eq(), Le(), and Ge(). The methods of IRange allow you to query the expressions and bounds of a ranged constraint and to change its bounds or replace the expression. In addition to this, the CPLEX methods Cplex.SetLinearCoef() and Cplex.SetLinearCoefs() allow you to edit the constraint expression for the special case of a linear expression, as supported by Cplex. Cplex is available to users of ILOG CPLEX.
