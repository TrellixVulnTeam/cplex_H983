Callback context

These macros are available in the CPLEX Callable Library (C API) to declare the context of a callback.

  • CPX_CALLBACKCONTEXT_CANDIDATE
    Generic callback at feasible solution candidate
  • CPX_CALLBACKCONTEXT_GLOBAL_PROGRESS
    Generic callback at global progress
  • CPX_CALLBACKCONTEXT_LOCAL_PROGRESS
    Generic callback at thread-local progress
  • CPX_CALLBACKCONTEXT_RELAXATION
    Generic callback at relaxation solution candidate
  • CPX_CALLBACKCONTEXT_THREAD_DOWN
    Generic callback at thread end
  • CPX_CALLBACKCONTEXT_THREAD_UP
    Generic callback at thread start
