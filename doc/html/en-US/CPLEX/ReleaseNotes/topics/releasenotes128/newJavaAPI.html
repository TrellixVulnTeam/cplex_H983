New features in the Java API of CPLEX V12.8

These release notes highlight new features in the Java API of in IBM CPLEX V12.8.

In the Java API of CPLEX V12.8, the new implementation of generic callbacks offers these new classes, interfaces, methods, and and enumerations (enum). They are documented in the reference manual of the Java API of CPLEX.
