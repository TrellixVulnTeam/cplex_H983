Release notes for IDE and OPL V12.8.0

These release notes highlight changes in the IDE and OPL V12.8.0. Please review these notes before using IDE and OPL V12.8.0.

Here are the release notes for IDE and OPL V12.8.0

New port for OPL on Z Linux

On Z Linux (S/390 Linux systems): a new port for OPL runtime, oplrun, C++ and Java APIs.

OPL available on Mac

The full OPL suite is available on Mac with the IDE, oplrun, the runtime and the C++/Java APIs

Updated settings panel

You can find more CPLEX/CPO settings directly from the IDE.

Java and .NET APIs updated

You can find new parameters for the engines.

Change in OPL API

IloCplex has moved from ilog.cplex to ilog.opl. This avoids possible confusion between the opl.jar and the cplex.jar.

Two-dimensional data supported

An alternative way of visualizing two-dimensional data using a matrix view is supported.

CES back in the IDE on Windows and Linux

From the IDE, you can call CPLEX Server to run your models.

Run seed action in the IDE

From the IDE or oplrun, you can now run your problem using CPLEX and CPO with different seeds to assess performance variability and to check its stability.
