CPXXgetlongannotationdefval and CPXgetlongannotationdefval

The routine CPXXgetlongannotationdefval/CPXgetlongannotationdefval accesses the default value of an annotation of a CPLEX problem object.

int CPXXgetlongannotationdefval( CPXCENVptr env, CPXCLPptr lp, CPXDIM idx, CPXLONG * defval_p )

int CPXgetlongannotationdefval( CPXCENVptr env, CPXCLPptr lp, int idx, long * defval_p )

Description

The routine CPXXgetlongannotationdefval/CPXgetlongannotationdefval accesses the default value of an annotation of a CPLEX problem object. The annotation must by of type double.

Arguments

Name Description
env A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
lp A pointer to a CPLEX problem object as returned by CPXXcreateprob/CPXcreateprob.
idx A integer indicating the index of the annotation for which to query the default value.
defval_p A pointer to a CPXLONG where to return the default value of the annotation.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example

status = CPXgetlongannotationdefval (env, lp, 0, &defval);
