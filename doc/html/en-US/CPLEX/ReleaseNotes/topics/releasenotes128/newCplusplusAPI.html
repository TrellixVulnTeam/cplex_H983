New features in the C++ API of CPLEX V12.8.0

These release notes highlight new features in the C++ API of in IBM CPLEX V12.8.0.

In the C++ API of CPLEX V12.8.0, the new implementation of generic callbacks offers these new classes, methods, structures (struct), type definitions (typedef), and enumerations (enum). They are documented in the reference manual of the C++ API of CPLEX.
