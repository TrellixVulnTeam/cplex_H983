CPXXstoreNEThandle and CPXstoreNEThandle

This routine stores a handle to a network model in a serializer.

int CPXXstoreNEThandle( CPXCENVptr env, CPXSERIALIZERptr s, CPXCNETptr net )

int CPXstoreNEThandle( CPXCENVptr env, CPXSERIALIZERptr s, CPXCNETptr net )

Description

This routine stores a handle to a network model in a serializer. Specifically, the routine stores the handle net in the specified serializer. The handle must be a member of env.

Arguments

env
The environment to which net belongs.
s
The serializer to which the handle is written.
net
The network handle to store.

Return

Returns 0 (zero) on success or an error code otherwise.
