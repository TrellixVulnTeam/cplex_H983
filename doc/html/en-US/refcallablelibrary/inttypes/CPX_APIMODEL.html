CPX_APIMODEL

API model selector.

#define CPX_APIMODEL configuration_dependent_value
This preprocessor definition selects which API model the routines in cplexx.h use for interaction with the CPLEX Callable Library (C API). Depending on the value set for this preprocessor definition, the types CPXDIM, CPXNNZ, and CPXCNT, designate different types themselves.
Table 1. Types of CPXDIM, CPXNNZ, and CPXCNT, depending on CPX_APIMODEL
  CPX_APIMODEL_SMALL CPX_APIMODEL_LARGE
CPXDIM CPXINT CPXINT
CPXNNZ CPXINT CPXLONG
CPXCNT CPXLONG CPXLONG
