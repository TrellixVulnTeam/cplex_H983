Accessibility in the online IBM Knowledge Center documentation

This section describes some of the accessibility features available when using the online IBM Knowledge Center documentation.

Using a screen reader with the IBM Knowledge Center

The IBM Knowledge Center has been tested using the screen reader JAWS 17.0 with Internet Explorer 11 and Mozilla Firefox 37.

Font sizes

The font sizes in the IBM Knowledge Center are set using relative font sizes. You can change them using functions available in your browser.

Keyboard shortcuts

Use the following key combinations to navigate the help system by keyboard. Note that there might be a slight variation in supported key combinations, depending on which browser you are using. Always check the browser help information if you find that the following combinations do not work.

  • To go to the next link, button or topic from inside a frame (page), press Tab.

  • To go to the previous link, button, or topic from inside a frame, press Shift+Tab.

  • To expand a node in the navigation tree, press the Right arrow.

  • To collapse a node in the navigation tree, press the Left arrow.

  • To move to the next node, press the Down arrow.

  • To move to the previous node, press the Up arrow.

  • To move to the beginning of the page, press Ctrl+Home.

  • To move to the end of the page, press Ctrl+End.

  • To go back, press Alt+Left arrow.

  • To go forward, press Alt+Right arrow.

  • To go to the next frame, press Ctrl+Tab.

  • To move to the previous frame, press Shift+Ctrl+Tab.

  • To print the current page or active frame, press Ctrl+P or Alt+Shift+P.

Alternate text for images

All images in the IBM Knowledge Center that convey additional information have alternate text. The alternate text can be read by a screen reader in place of the image.
