Integer types in the CPLEX Callable Library (C API) Reference Manual

These types of integers are available in the CPLEX Callable Library (C API).

  • CPXBYTE
    An 8-bit signed integer
  • CPXCNT
    Integer type for large counters.
  • CPXDIM
    Integer type for problem dimensions.
  • CPXINT
    A 32 bit, signed integer.
  • CPXLONG
    A 64-bit signed integer
  • CPXNNZ
    Integer type for nonzero counts.
  • CPXSHORT
    A 16-bit signed integer
  • CPXSIZE
    Signed size type.
  • CPXULONG
    A 64-bit unsigned integer
  • CPX_APIMODEL
    API model selector.
  • CPX_APIMODEL_LARGE
    Large data model selector.
  • CPX_APIMODEL_SMALL
    Small data model selector.
