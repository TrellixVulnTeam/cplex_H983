CPXERR_ILL_DEFINED_PWL

1213 ill defined piecewise linear function

#define CPXERR_ILL_DEFINED_PWL 1213

A piecewise linear function has been defined in a way that is not allowed. For example, it has an isolated point (a discontinuity at an anchor point); the breakpoints are not ordered; or the x coordinate of a breakpoint is +/- infinity.
