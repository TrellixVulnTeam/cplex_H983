Remote Object (Benders Algorithm) in the CPLEX Callable Library (C API)

These routines are available in the CPLEX Callable Library (C API) to support Benders algorithm and decompositions of a model in an application of the remote object.

  • CPXXbendersopt_async and CPXbendersopt_async
    Asynchronous version of CPXXbendersopt/CPXbendersopt.
  • CPXXbendersopt_join and CPXbendersopt_join
    Asynchronous version of CPXXbendersopt/CPXbendersopt.
  • CPXXbendersopt_multicast and CPXbendersopt_multicast
    Multicast version of CPXXbendersopt/CPXbendersopt.
