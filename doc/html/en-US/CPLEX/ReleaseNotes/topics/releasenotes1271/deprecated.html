Deprecated features CPLEX V12.7.1

These release notes announce deprecated features in IBM ILOG CPLEX V12.7.1. These deprecated features will be removed in a future release.

Deprecated UTF-16, UTF-32

CPLEX V12.7.1 does not guarantee support of UTF-16 nor UTF-32, the multi-byte encodings (also known as encoding pages). Indeed, CPLEX may remove support of UTF-16 and UTF-32 in a future release. For suggestions about a better choice of a multi-byte encoding, see the topic Selecting an encoding in the CPLEX User's Manual as well as documentation of the API string encoding switch and file encoding switch in the CPLEX Parameters Reference Manual.

Callable Library (C API)

The undocumented macro CPXoptimize has been deprecated and will be removed in a future release. Use the routine CPXXlpopt and CPXlpopt instead; the preferred routine is documented in the reference manual of the Callable Library (C API).

The undocumented routines CPXXparenv/CPXparenv and CPXXfreeparenv/CPXfreeparenv have been deprecated and will be removed in a future release.

C++ API

The class IloCplex::PresolveCallbackI has been deprecated and will be removed in a future release.

The macros to declare presolve callback functions have also been deprecated and will be removed in a future release:
  • ILOPRESOLVECALLBACK0
  • ILOPRESOLVECALLBACK1
  • ILOPRESOLVECALLBACK2
  • ILOPRESOLVECALLBACK3
  • ILOPRESOLVECALLBACK4
  • ILOPRESOLVECALLBACK5
  • ILOPRESOLVECALLBACK6
  • ILOPRESOLVECALLBACK7

Java API

The class IloCplex.PresolveCallback has been deprecated and will be removed in a future release.

.NET API

The class Cplex.PresolveCallback has been deprecated and will be removed in a future release.

Python API

The class Cplex.callbacks.PresolveCallback has been deprecated and will be removed in a future release.

The method cplex.terminate has been deprecated and will be removed in a future release.

The method SolveCallback.set_vectors has been deprecated and will be removed in a future release.

These methods have been deprecated and will be removed in a future release:
  • Cplex.solution.basis.get_col_basis. Use Cplex.solution.basis.get_basis instead.
  • Cplex.solution.basis.get_row_basis. Use Cplex.solution.basis.get_basis instead.
  • Cplex.start.set_basis. Use Cplex.start.set_start instead.
These methods have also been deprecated and will be removed in a future release:
  • Cplex.linear_constraints.advanced.add_lazy_cuts. Use add_lazy_constraints instead.
  • Cplex.linear_constraints.advanced.free_lazy_cuts. Use free_lazy_constraints instead.

These Python examples have been updated to remove deprecated methods and to use instead the recommended methods:

  • lpex2.py
  • lpex6.py
  • lpex7.py
  • qpex2.py
