CPXMI_EPGAP_OBJOFFSET

1037 Solution optimal within relative optimality tolerance of %g, but non optimal if objective offset of value %g is removed

#define CPXMI_EPGAP_OBJOFFSET 1037
The optimal solution is optimal within the specified relative optimality tolerance, but it is not optimal within the same tolerance if the objective offset is removed. Consider removing the objective offset or decreasing the relative optimality tolerance CPX_PARAM_EPGAP (defined in cplex.h) to possibly find a more accurate optimal solution.
