Conversion notes for CPLEX V12.7.1

Changes in features of CPLEX V12.7.1

These release notes highlight changes of features in IBM ILOG CPLEX V12.7.1. Please review these notes before using IBM ILOG CPLEX V12.7.1.

Name of example changed

Previous versions of CPLEX distributed an example titled transport.cpp to demonstrate in the C++ API how to apply piecewise linear (PWL) constraints to solve a logistics problem with varying cost coefficients. The example remains the same, but its name is now ilotransport.cpp in keeping with the naming convention of other examples in the C++ API of CPLEX.
