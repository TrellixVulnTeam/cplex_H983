Remarks

Branch directions can also be used with the method Cplex.SetDirection and Cplex.SetDirections to select the child node to evaluate first after branching on a specific variable. There are three settings for this purpose:

  • Up: process the child where the lower bound of the variable has been tightened first.
  • Down: process the child where the upper bound of the variable has been tightened first.
  • Global: process the child first that the MIP optimizer would have ordinarily chosen.

Branch directions can also be used with the methods Cplex.BranchCallback.MakeBranch to specify branching direction and bounds for an array of variables. The settings have slightly different effects in Cplex.BranchCallback.MakeBranch.

  • Up: set the lower bound of the corresponding variable to the specified bound.
  • Down: set the upper bound of the corresponding variable to the specified bound.
