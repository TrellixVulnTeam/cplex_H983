CPXCNT

Integer type for large counters.

typedef configuration_dependent_type CPXCNT
CPXCNT is the integer type that cplexx.h uses to pass potentially large counts, such as node counts, iteration counts, and so forth, to the CPLEX Callable Library (C API) or to read such data from it. The actual type of CPXCNT depends on the current setting of CPX_APIMODEL.
