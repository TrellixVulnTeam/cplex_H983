CPXXaddfpdest and CPXaddfpdest

The routine CPXXaddfpdest/CPXaddfpdest adds a file to the list of message destinations for a channel.

int CPXXaddfpdest( CPXCENVptr env, CPXCHANNELptr channel, CPXFILEptr fileptr )

int CPXaddfpdest( CPXCENVptr env, CPXCHANNELptr channel, CPXFILEptr fileptr )

Description

The routine CPXXaddfpdest/CPXaddfpdest adds a file to the list of message destinations for a channel. The destination list for all CPLEX-defined channels is initially empty.

The routine CPXXaddfpdest/CPXaddfpdest consults the parameter known as the file encoding switch (CPXPARAM_Read_FileEncoding) to determine the encoding (such as UTF-8) to associate with the file when it adds a file to the list of destinations for a channel. This encoding persists, regardless of later changes in the parameter. This convention of persistence of the initial encoding means that the user can associate different encodings with the writing of different files.

Arguments

env
A pointer to the CPLEX environment as returned by CPXXopenCPLEX/CPXopenCPLEX.
channel
A pointer to the channel for which destinations are to be added.
fileptr
A pointer to the file to be added to the destination list. Before calling this routine, obtain this pointer with a call to CPXXfopen/CPXfopen.

Return

The routine returns 0 (zero) if successful and nonzero if an error occurs.

Example

CPXaddfpdest (env, mychannel, fileptr);

See lpex5.c in the CPLEX User's Manual.
