New callback design in CPLEX V12.8.0

These release notes highlight newly redesigned callbacks in IBM CPLEX V12.8.0.

CPLEX V12.8.0 offers a new suite of callbacks that are not subject to the limitations on existing control callbacks (such limitations as incompatibility with dynamic search). In fact, these new callbacks represent a fully redesigned implementation of callbacks in CPLEX.

For more about this fresh approach to callbacks, see the new topics Generic callbacks in the CPLEX User's Manual. See also the announcements in these release notes of new elements (routines, classes, methods, enumerations) in the APIs of CPLEX:
  • New features in the Callable Library (C API) of CPLEX V12.8
  • New features in the C++ API of CPLEX V12.8.0
  • New features in the Java API of CPLEX V12.8
  • New features in the .NET API of CPLEX V12.8
  • New features in the Python API of CPLEX V12.8
These new elements are documented in the corresponding reference manuals.

CPLEX V12.8.0 introduces new error codes supporting the new generic callbacks and their contexts. See the announcement New error codes in CPLEX V12.8.0 in these release notes for a complete list of new error codes.
