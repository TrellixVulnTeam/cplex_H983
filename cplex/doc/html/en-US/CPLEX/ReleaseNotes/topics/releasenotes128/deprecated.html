Deprecated features CPLEX V12.8.0

These release notes announce deprecated features in IBM CPLEX V12.8.0. These deprecated features will be removed in a future release.

Parameters

The parameter limit on nodes explored when CPLEX solves a subMIP has been deprecated. It will be removed in a future release. Use instead the new parameter limit on nodes explored when CPLEX solves a subMIP.

Interactive Optimizer

The following commands of the Interactive Optimizer have been deprecated. They will be removed from a future release.
  • set output dialog
  • set output errors
  • set output logonly
  • set output results
  • set output warnings
  • set mip limits submipnodelim

    Use set mip submip nodelimit instead.

Callable Library (C API)

Deprecated error codes

The error code CPXERR_CHILD_OF_CHILD has been deprecated.

The error code CPXERR_CANT_CLOSE_CHILD has been deprecated.

The only routine that triggered these two errors, CPXfreeparenv, has been removed, so these errors cannot occur in the future.

Deprecated routines

The following routines have been deprecated in CPLEX V12.8. They will be removed in a future release.
  • CPXXsetlogfile/CPXsetlogfile

    Instead, use CPXXsetlogfilename and CPXsetlogfilename.

  • CPXXfopen/CPXfopen
  • CPXXfclose/CPXfclose
  • CPXXaddfpdest/CPXaddfpdest
  • CPXXdelfpdest/CPXdelfpdest
  • CPXXfputs/CPXfputs

For example, these lines of code have been deprecated:

 
FILE *out = CPXfopen(...);   //deprecated
CPXaddfpdest(env, channel, out); //deprecated

Replace such deprecated code with lines like these instead:

 
                   static void CPXPUBLIC
                   msgfun (void file, char const *text) { fputs(text, file); }
                   FILE *out = fopen(...);
                   CPXaddfuncdest(env, channel, out, msgfun);
