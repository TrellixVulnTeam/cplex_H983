Annotating Models for Benders Decomposition in the CPLEX Callable Library (C API)

These macros are available in the CPLEX Callable Library (C API) for you to annotate your model. An annotation associates an ordered pair of key and value with an object in your model. You can use these annotations, for example, to specify blocks in a Benders decomposition of your model.

For more information about annotations and Benders decomposition, see these topics in the CPLEX User's Manual:
  • Benders algorithm and
  • Annotated decomposition for Benders algorithm.
  • CPX_BENDERS_ANNOTATION
    The name of the Benders annotation
  • CPX_BENDERS_MASTERVALUE
    Default value for the Benders master partition
