End of file in LP file format

CPLEX conforms to these rules about end of file (EOF) in LP file format.

In LP file format, the file must end with the keyword end in any combination of upper- and lowercase characters, alone on a line, when it is created with the enter command. This word is not required for files that are read in to CPLEX, but it is a good practice to use it. Files that have been corrupted can frequently be detected by a missing last line.
