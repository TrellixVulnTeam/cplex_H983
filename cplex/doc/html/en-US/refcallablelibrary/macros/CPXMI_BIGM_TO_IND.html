CPXMI_BIGM_TO_IND

1041 In %s, where %s has a coefficient of %g, consider using an indicator

#define CPXMI_BIGM_TO_IND 1041
An indicator constraint models a condition where the value of a binary variable controls the activity of a linear constraint. The linear constraint mentioned in this warning could be reformulated as an indicator where the binary variable mentioned implies the rest of the constraint.
